# NeuroScan

Легка демо-система для аналізу МРТ-зображень головного мозку з використанням нейромережі ResNet18.

## Що є в проєкті

- `backend/` — FastAPI-сервер для прогнозів з моделлю `brain_tumor_resnet18.pt`.
- `ml/` — модельні утиліти, трансформації та Grad-CAM.
- `src/` — фронтенд-інтерфейс для завантаження зображень, аналізу та відображення теплової карти.
- `models/` — збережені ваги моделей та метрики.
- `index.html` — односторінковий UI.

## Запуск

1. Встановіть залежності:
   ```powershell
   pip install -r requirements.txt
   ```

2. Запустіть бекенд:
   ```powershell
   python run_backend.py
   ```

3. Відкрийте `index.html` у браузері або запустіть простий локальний сервер.

## Як користуватися

- Завантажте МРТ-зображення.
- Натисніть «Аналіз» для отримання прогнозу та Grad-CAM.
- Історія останніх вимірювань зберігається у браузері.
- Є можливість експортувати та імпортувати історію.

## Утиліти

- CLI для локального інференсу: `backend/infer_image.py`
- Grad-CAM логіку та накладання реалізовано у `ml/visualize.py`

## Зауваження

- Проєкт розрахований на локальне тестування.
- Для коректної роботи бекенду має бути встановлена `scipy` разом з `torch` та `fastapi`.
